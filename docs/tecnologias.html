Decisiones técnicas clave

Next.js 16

Usa proxy.ts en lugar de middleware.ts. La función se llama proxy, no middleware.

shadcn/ui

Usa @base-ui/react. Sin asChild. Para botones con Link usar buttonVariants() + <Link>.

Supabase

Siempre usar Session Pooler (puerto 5432). Direct Connection falla en IPv4. La región recomendada es São Paulo.

Railway

Root Directory = backend/FiltCar.Api. Puerto de escucha: 8080. AllowedOrigins separados por coma.
